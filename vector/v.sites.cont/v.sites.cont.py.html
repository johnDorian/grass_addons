DESCRIPTION

v.sites.cont Calculates the contributing area of each point within a vector.

This module will calculate the contributing area for each point in a vector layer. In addition, the addon will also calculate the statistics of raster maps within the contributing area.

Caluculation of the contributing area is performed using the teh flow direction map produced using r.watershed and is required to determine the contributing area which is undertaken using the r.wateroutlet modulde.

It is recommended that the points within the vector layer are snapped to the stream using the r.stream.snap module to ensure the points are aligned correctly for the calculation. However, this is not required and the contributing area can still be determined for locations not aligned with streams.

This module will calculate the contributing area for each point, and if additional raster maps are provided, it will use the contributing area as a raster to mask the other raster maps and calculate the statistics for these maps.

This module calculates statistics using two different methods:

Continous maps This option is for maps with continuous values and will calculate the minimum, mean, etc for this map and add columns to the inputted vector map accordingly.

Categorical maps This option will calculate the area statistics for these maps. This is useful for landclass style maps. It will create a column in the vector layer for each type of unique value in the map and determine the area within of each type within the sub-catchment.
